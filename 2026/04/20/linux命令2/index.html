一、Linux 系统核心操作

（一）进程管理

1. 查看进程信息

bash

通过ps命令查看Linux系统中的进程信息

-e：显示全部的进程

-f：以完整格式形式展示信息

关闭进程，

2. 实时监控系统资源

top命令查看CPU、内存使用情况

交互操作（运行过程中）：

h：显示帮助画面
c：显示完整命令
P：根据CPU使用百分比排序
M：根据内存占用大小排序
T：根据时间/累计时间排序
E：切换顶部内存显示单位
e：切换进程内存显示单位
l：切换显示平均负载和启动时间信息
i：不显示闲置或无用的进程
t：切换显示CPU状态摘要
m：切换显示内存摘要

常用参数：

-p：只显示某个进程信息
-d：设置刷新时间，默认5s
-c：显示产生进程的完整命令，默认进程名
-n：指定刷新次数，例如 top -n 3 3次后退出
-b：以批处理模式运行
-u：查找特定用户启动的进程  

3. 终止进程

kill：关闭进程（需指定进程号）
-q：强制关闭进程
kill -q [进程号]  

（二）用户与用户组管理

1. 用户组操作

创建用户组

groupadd [用户组名]

删除用户组

groupdel [用户组名]  

2. 用户操作

useradd创建用户

-g：指定用户的组，不指定则创建组自动加入
-d：指定用户Home路径，不指定则默认在/home/用户名
useradd [-g -d] [用户名]

删除用户

-r：删除用户的Home目录（默认不删除）
userdel [-r] [用户名]

查看用户所属组

id [用户名]

将指定用户加入用户组

usermod -aG [用户组] [用户名]  

3. 用户切换

切换用户（普通用户切换需密码，root切换无需密码）

su [-] [用户名]

退出当前登录用户（等效 Ctrl+D）

exit

临时以root身份执行命令

sudo [命令]

配置sudo认证（切换root用户执行visudo，文件末尾添加）

格式：用户名 ALL=(ALL) NOPASSWD: ALL

保存退出：wq

 

（三）文件权限与目录操作

1. 权限修改

chmod：修改文件/文件夹权限

语法：chmod [-R] 权限 文件/文件夹

-R：对文件夹内全部内容应用相同权限

权限符号：u=用户( rwx )、g=组( rwx )、o=其他( rwx )

权限数字：0(无权限)、1(仅执行)、2(仅写)、3(写+执行)、4(仅读)、5(读+执行)、6(读+写)、7(全部权限)

chmod [-R] 751 test.txt  

2. 所属主/组修改

chown：修改文件/文件夹所属用户和用户组

语法：chown [-R] [用户][:用户组] 文件/文件夹

chown [-R] [用户][:用户组] [文件/文件夹]  

3. 目录查看

查看目录列表（等效 ls -l）

ll

查看磁盘使用情况

df [-h]

查看CPU、磁盘相关信息

iostat [-x] [刷新间隔] [刷新次数]  

（四）网络与服务管理

1. 网络测试与下载

ping：检查网络连通性

-c：指定检查次数

ping [-c num] [IP/主机名]

wget：非交互式文件下载器

-b：后台下载
wget [-b] [URL]

curl：发送HTTP网络请求

-O：下载文件
curl [-O] [URL]  

2. 端口管理

查看端口占用情况

netstat -anp | grep [端口号]

nmap：查看所有端口占用情况

nmap [IP]

端口说明：

注册端口：1024-49151（可随意使用）
动态端口：49152-65535（临时分配）  

3. 服务管理

systemctl：管理Linux系统服务

常用命令：start(启动)、stop(停止)、status(查看状态)、enable(开机自启)、disable(开机禁用)

systemctl [start/stop/status/enable/disable] [服务名]  

（五）压缩与解压

tar：压缩/解压

常用参数：

-c：创建压缩文件
-v：显示进度
-x：解压
-f：指定压缩/解压文件
-z：gzip模式
-C：指定解压目录
压缩：tar [-zcvf] 压缩包名 目标文件/文件夹
tar -zcvf test.tar.gz test/

解压：tar [-zxvf] 压缩包名 [-C 解压目录]

tar -zxvf test.tar.gz -C ./test/

unzip：解压zip文件

-d：指定解压位置
unzip [-d] [解压目录] [压缩包]  

（六）环境变量与系统命令

1. 环境变量

查看当前系统环境变量

env

临时设置环境变量

export 变量名=变量值

使配置文件生效

source [配置文件]

示例：添加路径到PATH

export PATH=$PATH:/usr/local/bin  

2. 系统时间

date：查看/设置系统时间

语法：date [-d] [+格式化字符串]

-d：按指定字符串显示日期（用于日期计算）

格式符：%Y(年)、%y(年份后两位)、%m(月)、%d(日)、%H(小时)、%M(分钟)、%S(秒)、%s(1970-01-01 00:00:00 UTC秒数)

示例：显示明天日期

date -d “+1 day” +%Y%m%d  

3. 历史命令

history：查看历史执行命令

history

快捷操作：

Ctrl+R：搜索历史命令
Ctrl+A：跳到命令开头
Ctrl+E：跳到命令结尾
Ctrl+左/右箭头：向左/右跳一个单词
Ctrl+L：清屏（等效 clear）  

（七）Vim 编辑器

1. 工作模式

模式切换：

命令模式 -> 输入模式：i(当前位置)、I(行首)、a(当前后)、A(行尾)、o(下一行)、O(上一行)
输入模式 -> 命令模式：ESC键
命令模式 -> 底线命令模式：:键
底线命令模式 -> 命令模式：ESC键  

2. 命令模式常用操作

光标移动：

k(上)、j(下)、h(左)、l(右)
$：行尾、0：行首
pgUp：向上翻页、pgDn：向下翻页
gg：跳到首行、G：跳到尾行
复制/删除/粘贴：
dd：删除当前行
ndd：删除当前行及向下n行
yy：复制当前行
nyy：复制当前行及向下n行
p：粘贴复制内容
u：撤销修改
Ctrl+R：反向撤销修改
全删/删除：
dG：从当前行开始向下全删
dgg：从当前行开始向上全删
d$：删除到本行结尾
d0：删除到本行开头

搜索：/

n：继续向下搜索、N：继续向上搜索  

3. 底线命令模式常用操作

保存/退出：

:w：保存、:q：退出、:wq：保存并退出、:q!：强制退出

显示设置：

:set nu：显示行号、:set nonu：取消行号
:set paste：粘贴模式（避免格式错乱）
